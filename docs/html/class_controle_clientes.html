Referência da Classe ControleClientes

A classe ControleClientes gerencia operações relacionadas aos clientes no sistema. Ela mantém uma lista de objetos Cliente e fornece métodos para cadastrar, remover, pesquisar e gerar relatórios sobre os clientes.
