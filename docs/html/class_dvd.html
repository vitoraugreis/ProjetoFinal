Referência da Classe Dvd

A classe Dvd é uma representação de um DVD no contexto do sistema de controle de mídia. Ela herda características da classe base Midia e adiciona funcionalidades específicas para DVDs.

O método 'imprimirInformacoes(bool unidades)' é um método específico para DVDs que imprime informações sobre o DVD, incluindo o código, título e, opcionalmente, o número de unidades disponíveis. Útil para visualização e relatórios.
