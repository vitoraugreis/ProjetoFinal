Projeto Locadora Documentação
Projeto Locadora ver 2.0

Uma vídeo-locadora entrou em contato com nosso grupo pedindo a criação de um sistema que possibilite um melhor controle de toda a loja. Este sistema deveria ser capaz de:

- Gerenciar o estoque de filmes;

- Gerenciar os clientes da vídeo-locadora;

- Gerenciar os aluguéis de filmes;

- Calcular automaticamente o quanto o cliente deve pagar na devolução dos filmes.



Objetivo e Funcionalidades

O sistema tem como propósito principal gerenciar o acervo de filmes da locadora, contemplando DVDs e Fitas de Vídeo, categorias específicas para DVDs, e definindo valores de locação de acordo com a mídia e categoria.

Foi desenvolvido com uma abordagem de orientação a objetos e oferece funcionalidades como o cadastro de filmes com código, título, unidades disponíveis, e categorias específicas para DVDs. O valor da locação varia conforme a mídia (DVD ou Fita) e a categoria do DVD. Adicionalmente, há tratamento especial para fitas não rebobinadas.

O sistema gerencia clientes, registrando nome e CPF. Possui operações para inserir novos clientes, remover clientes, além de funcionalidades para pesquisar e imprimir relatórios de clientes.

O cálculo do valor a ser pago na devolução dos filmes é realizado com base na mídia (DVD ou Fita), categoria do DVD, e dias de locação. São aplicadas diferentes fórmulas de preço para DVDs de lançamento, estoque e promoção, bem como para fitas, incluindo uma multa de 2 reais caso a fita seja devolvida sem estar rebobinada.


A linguagem utilizada para desenvolvimento foi C++.


Divisão do projeto


No projeto da locadora de filmes em C++, a estrutura foi organizada de maneira modular, dividindo o sistema em componentes distintos para facilitar a manutenção e compreensão do código. Cada componente representa uma funcionalidade específica e contém classes relacionadas para garantir uma organização eficiente do código-fonte.
