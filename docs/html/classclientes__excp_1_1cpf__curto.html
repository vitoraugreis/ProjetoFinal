Referência da Classe clientes_excp::cpf_curto

A classe cpf_curto é uma classe de exceção definida no contexto do sistema de controle de clientes. Essa exceção é lançada quando ocorre uma tentativa de cadastrar um cliente com um CPF que possui um número de dígitos menor do que o esperado, neste caso, um CPF com menos de 11 dígitos.

A exceção cpf_curto é utilizada para sinalizar que a validação do CPF falhou devido ao comprimento insuficiente do número. Isso indica que o CPF fornecido para o cadastro não atende aos requisitos mínimos de um CPF válido.
