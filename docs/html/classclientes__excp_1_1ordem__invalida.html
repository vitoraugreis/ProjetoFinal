Referência da Classe clientes_excp::ordem_invalida

A classe ordem_invalida é uma classe de exceção definida no contexto do sistema de controle de clientes. Essa exceção é lançada quando ocorre uma tentativa de ordenar os clientes com base em uma ordem que não é reconhecida ou suportada pelo sistema.

A exceção ordem_invalida é utilizada para sinalizar que a operação de ordenação não pode ser concluída porque a ordem especificada é inválida. No contexto específico, a ordenação dos clientes pode ser realizada com base no CPF (C) ou no nome (N).
