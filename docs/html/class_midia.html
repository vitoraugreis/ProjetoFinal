Referência da Classe Midiaabstract

A classe Midia é uma classe abstrata que serve como base para diferentes tipos de mídia, como DVDs e fitas. Ela contém atributos e métodos comuns a todas as mídias, e outras classes específicas de mídia herdam dela. Possui os métodos reponsáveis por diminuir ou aumentar as unidades disponíveis e imprimir informações que é um método virtual.

A classe verifica se a categoria fornecida durante a criação da mídia é válida, lançando uma exceção categoria_invalida se não for.

#include <Midia.hpp>
